## 1. 개요

농협사료 사무관리직을 위한 「생성형 AI 기반 사무관리 업무혁신 실무」 1일 8차시 과정의
학습사이트를 개발했다. 무료 AI 도구(ChatGPT·Claude·Gemini·Perplexity·Google Sheets·Docs)로
문서 작성 → 자료 조사 → 시장 분석 → 기획 → 보고서 자동화 → 데이터 분석 → 의사결정 지원의
업무 흐름을 실습하도록 구성했다.

기존 **bulid**(건설기계 AI역량강화) 프로젝트를 베이스로 재사용하여, 2권×6일 구조를
**단일 과정 × 8차시** 구조로 재편하고 농협(NH) 아이덴티티로 리브랜딩했다.

## 2. 기술 스택

- React 18 + Vite 5 + React Router 6
- Tailwind CSS (Pretendard 폰트, Font Awesome 6)
- Supabase (구글·카카오 OAuth, 학습 진도 동기화)
- GitHub Actions → GitHub Pages 자동 배포 (커스텀 도메인, base `'/'`)

## 3. 주요 작업

### 3-1. 리브랜딩 (bulid → 농협사료)
- **컬러**: 딥 네이비+옐로우 → **농협 딥 그린(#12522e) + 농협 골드(#fec20d)**
  - 골드값은 제공받은 공식 CI 파일(심볼마크.ai)에서 실제 색을 추출해 정확히 반영
- **로고**: 안전모 아이콘 → **농협 공식 심볼마크(노란 새싹) 추출본**
  - 제공된 `시그니춰.ai`(PostScript)를 Ghostscript로 렌더 → PIL로 심볼 자동 크롭 →
    투명 PNG(`public/nh-symbol.png`) 및 파비콘(`public/favicon.png`) 생성
- 헤더/로그인모달 로고, index.html 메타·OG, `SITE_KEY`, Supabase 진도 테이블
  (`nonghyupsaryo_progress`), localStorage 키 전면 교체

### 3-2. 콘텐츠 (8차시 커리큘럼)
- `content.json`을 제안서 기반 8차시로 신규 작성 (블록 렌더링: 학습목표·표·프롬프트박스·실습·산출물)
  1. 프롬프트 엔지니어링 이해
  2. 사무관리 업무별 AI 활용 구조 이해
  3. AI 기반 시장분석 및 트렌드 도출
  4. AI 활용 전략 기획 및 아이디어션
  5. GPT 활용 기획 문서 및 보고서 자동화
  6. AI 기반 데이터 분석
  7. AI 활용 의사결정 지원 시스템 구축
  8. 종합 실습 및 현업 적용 계획 수립
- `dayplans.js`: 08:00~17:00 단일일 8차시 타임테이블(점심 1시간 제외)로 재작성

### 3-3. 도구·실습·부록 데이터 재작성 (병렬 서브에이전트)
- `tools.js`: Genspark 제거, **Google Sheets·Docs 추가** → 6종 도구, 사무관리 맥락으로 전면 재작성
- `labs.js`: 단일 `course` 볼륨, 8차시 21개 실습(5요소 프롬프트)
- `appendix.js`: 업무별 프롬프트 7카테고리 21개(이메일·보고서·요약·조사·데이터·공지·의사결정)
- `promptLab.js`: SCORE 채점 엔진 유지, 예시만 사무관리로 교체
- `about.js`: 제작 의도·강사·회사 소개 재작성

### 3-4. 컴포넌트/페이지 단일과정 적응

## 5. 배포 메모

- GitHub Actions(`deploy.yml`) push 트리거 자동 배포
- 신규 Pages는 legacy로 시작될 수 있어 **build_type=workflow(Actions)** 전환 필요
- 커스텀 도메인이므로 vite `base: '/'` (자산 404 방지)
- Supabase 진도 동기화는 `nonghyupsaryo_progress` 테이블 필요(없어도 로컬 저장으로 graceful degrade)
